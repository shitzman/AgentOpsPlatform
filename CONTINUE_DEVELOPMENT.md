# Continue Development Prompt

Use this prompt for normal implementation sessions after the project has been bootstrapped.

```text
阅读：

README.md
AGENTS.md
CLAUDE.md
ROADMAP.md
TASKS.md
ARCHITECTURE.md

理解当前项目状态。

只完成 TASKS 中第一项未完成任务。

完成后：

更新 TASKS
更新 CHANGELOG
如果架构发生变化，同步更新 ARCHITECTURE。

不要实现未计划功能。
保持代码简洁。
遵循项目编码规范。

输出：

1. 本次完成内容
2. 修改文件
3. 下一步建议
```
